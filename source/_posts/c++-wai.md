# 一、二进制转换八进制

## 1. 二进制数与八进制数的对应关系

再选系具体方法之前，先记一下二进制数与八进制数之间的对应关系：


| 二进制 | 八进制 |
| :----: | :----: |
|  000  |   0   |
|  001  |   1   |
|  010  |   2   |
|  011  |   3   |
|  100  |   4   |
|  101  |   5   |
|  110  |   6   |
|  111  |   7   |

## 2.转换方法

### 二进制→八进制

在二进制转换八进制时，我们采用**取三合一法**
即从二进制的小数点为分界点，向左（或向右）每三位取成一位，分好组以后，对照二进制与八进制数的对应表（上面的表），将三位二进制按权相加，得到的数就是一位八进制数，然后按顺序排列，小数点的位置不变，最后得到的就是一个八进制数。
**举例：将二进制101000100.001110011转化为八进制数**
![完整转换](https://img1.imgtp.com/2023/07/15/CTQHmX6y.png)
在向左（或向右）取三位时，取到最高位（最低位）如果无法凑足三位，就可以在小数点的最左边（或最右边）补0，帮助进行换算
**举例：101 011 00.01可以补足为010 101 100.010**
![补零法](https://img1.imgtp.com/2023/07/15/7kuPb01n.png)

### 八进制→二进制

八进制转为二进制，反过来啦，方法是一分三，这个更简单一些，即一个八进制数分成三个二进制数，用三位二进制按权相加，最后得到二进制，小数点位置不变。
**例如：7534.21转换为二进制**
![2-8 001.png](https://img1.imgtp.com/2023/07/15/iu6zZ0eP.png)

## 转换方法

### 二进制→十六进制

二进制转换成十六进制的方法是，**取四合一**法，即从二进制的小数点为分界点，向左（或向右）每四位取成一位，如下图所示。
**举例：1011 1001 1011 . 1001转换16进制**
![2-16 001.png](https://img1.imgtp.com/2023/07/15/RWJQiTkb.png)
注意16进制的表示法，用字母H后缀表示，比如BH就表示16进制数11；也可以用0X前缀表示，比如0X23就是16进制的23.直观表示法如下所示。
<big>B<u>H</u></big> 中的**H**就是后缀
<big><u>0X</u>53</big> 中的**0X**就是后缀
这里需要注意的是，在向左（或向右）取四位时，取到最高位（最低位）如果无法凑足四位，就可以在小数点的最左边（或最右边）补0，进行换算，如下图所示
![2-16 002.png](https://img1.imgtp.com/2023/07/15/JDyt71Aj.png)

### 十六进制→二进制

下面看看将16进制转为二进制，反过来啦，方法就是一分四，即一个十六进制数分成四个二进制数，用四位二进制按权相加，最后得到二进制，小数点依旧就可以啦。如下图所示。
**举例：BF4B.5转换为十六进制**
![16-2 001.png](https://img1.imgtp.com/2023/07/15/LcjY1GOn.png)

## 注意事项

* 二进制转十六进制与二进制转八进制类似。
* 十六进制转二进制与八进制转二进制类似，请相互融合学习，达到融会贯通。
