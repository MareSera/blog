## 定义
**指针**：存储的数据地址（地址都是十六进制）
具体的十六进制参考[二进制-八进制-十六进制的转换方法](https://www.maresera.top/2023/07/14/c++-wai/)
## 声明
声明变量a  `int a=3;`  声明指针变量p  `int *p=&a`
**带`*`表明是指针类型的变量***
## 输出
输出变量p或&a会获得指向变量所占内存的第一个字节的编号
以上面声明的变量a和p为例
`cout<<p<<' '<<&a;`会获得
`0x63fe14 0x63fe14`
### 注意
`cout<<p<<' '<<&a;`中如果将`p`改成`*p`那么输出的将会是这个指针变量所指向的变量的值
也就是执行上方语句会输出以下结果
`3 0x63fe14`
### 程序
#### 程序
```c++
#include<bits/stdc++.h>
using namespace std;
int main() {

    int a[10]={1,2,3,4,5,6,7,8,9,0};
    int *p=a;
    for(int i=0;i<=9;i++){
        cout<<p<<" "<<*p<<endl;
        p++;
    }

     return 0;
}
```
#### 输出
```
0x63fde0 1
0x63fde4 2
0x63fde8 3
0x63fdec 4
0x63fdf0 5
0x63fdf4 6
0x63fdf8 7
0x63fdfc 8
0x63fe00 9
0x63fe04 0
```
仔细观察，分配的地址结尾分别是0、4、8、c，可以发现，每个都+4，这是因为`int`类型每个占4个字节（十六进制）同理，`int`改为`double`以后会变成下面这样
```
0x63fdc0 1
0x63fdc8 2
0x63fdd0 3
0x63fdd8 4
0x63fde0 5
0x63fde8 6
0x63fdf0 7
0x63fdf8 8
0x63fe00 9
0x63fe08 0
```
每个都+8，这是因为`double`类型每个占8个字节
